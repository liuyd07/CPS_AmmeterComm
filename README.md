This program mainly read electric varables from intelligent ammeters. This version is simple version, which only read current, voltage, power and power factor these 4 varables.
This program has also achieved read process from various number of ammeters, and change of varables at any time.
本程序主要是从智能电表中读取电学量。该版本是简化测试版，只实现对于电压、电流、视在功率、功率因数四个变量的读取。实现了对于任意多个的电表的数据读取。可以随时设置读取的变量。
